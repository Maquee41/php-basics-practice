# Модуль 04 - Массивы в PHP

## Что нужно изучить?

- Виды массивов
- Хранение структурированной информации с помощью массивов
- Основные функции для работы с массивами

### Простой массив

```php
$pets = ['кот', 'собака', 'птичка'];
```

### Кастомные индексы

```php
$keyboardsKeys = [
    27 => 'Esc',
    13 => 'Enter',
    32 => 'Space',
];
```

```php
$otherKeysArray = [
    3 => 'first',   // index: 3
    'second',       // index: 4
    'third'         // index: 5
];
```

### Ассоциативные массивы

```php
$car = [
    'name' => 'Грузовик',
    'engine' => 'Дизель',
    'year' => '2019',
];
```

### Удаление элемента из массива

```php
unset($array[0]);
unset($array['name']);
```
